5.3 Summarizing and Computing Descriptive Statistics¶

  • Pandas objects are equipped with a set of common mathematical and statistical methods.
  • Compared with the similar methods found on NumPy arrays, they have built-in handling for missing data.
In [1]:
import numpy as np
import pandas as pd
In [2]:
df = pd.DataFrame([[1.4, np.nan], [7.1, -4.5],[np.nan, np.nan], [0.75, -1.3]],\
             index=['a', 'b', 'c', 'd'],columns=['one', 'two'])
df
Out[2]:
one two
a 1.40 NaN
b 7.10 -4.5
c NaN NaN
d 0.75 -1.3

Calling DataFrames sum method returns a Series containing column sums:

In [3]:
df.sum()
Out[3]:
one    9.25
two   -5.80
dtype: float64

Passing axis='columns' or axis=1 sums across the columns instead:

In [4]:
df.sum(axis='columns')
Out[4]:
a    1.40
b    2.60
c    0.00
d   -0.55
dtype: float64

NA values are excluded unless the entire slice (row or column in this case) is NA. This can be disabled with the skipna option:

In [5]:
df.mean(axis='columns', skipna=False)
Out[5]:
a      NaN
b    1.300
c      NaN
d   -0.275
dtype: float64

See Table 5-7 for a list of common options for each reduction method.¶

Some methods, like idxmin and idxmax, return indirect statistics like the index value where the minimum or maximum values are attained:

In [6]:
df.idxmax()
Out[6]:
one    b
two    d
dtype: object

Other methods are accumulations:

In [7]:
df.cumsum()
Out[7]:
one two
a 1.40 NaN
b 8.50 -4.5
c NaN NaN
d 9.25 -5.8

describe producing multiple summary statistics in one shot:

In [8]:
df.describe()

On non-numeric data, describe produces alternative summary statistics:

In [9]:
obj = pd.Series(['a', 'a', 'b', 'c'] * 4)

dtype: object

1. Correlation and Covariance¶

Some summary statistics, like correlation and covariance, are computed from pairs of arguments. Finance using the add-on pandas-datareader package.

In [10]:
# !conda install pandas-datareader
In [12]:
import pandas_datareader.data as web
from pandas.util.testing import assert_frame_equal
all_data = {ticker: web.get_data_yahoo(ticker) for ticker in ['AAPL', 'IBM', 'MSFT', 'GOOG']}
In [15]:
price = pd.DataFrame({ticker: data['Adj Close'] for ticker, data in all_data.items()})
volume = pd.DataFrame({ticker: data['Volume'] for ticker, data in all_data.items()})

Now compute percent changes of the prices¶

In [17]:
returns = price.pct_change() * 100
returns.tail()
Out[17]:

Date

The corr method of Series computes the correlation of the overlapping, non-NA, aligned-by-index values in two Series. Relatedly, cov computes the covariance:

In [18]:
returns['MSFT'].corr(returns['IBM'])
Out[18]:
0.593161817067868
In [19]:
returns['MSFT'].cov(returns['IBM'])
Out[19]:
1.6660175252734752

We can also select these columns using more concise syntax:

In [20]:
returns.MSFT.corr(returns.IBM)
Out[20]:
0.593161817067868

DataFrames corr and cov methods, on the other hand, return a full correlation or covariance matrix as a DataFrame, respectively

In [21]:
returns.corr()
Out[21]:
AAPL IBM MSFT GOOG
AAPL 1.000000 0.527378 0.716384 0.646162
IBM 0.527378 1.000000 0.593162 0.526100
MSFT 0.716384 0.593162 1.000000 0.751614
GOOG 0.646162 0.526100 0.751614 1.000000

Using DataFrames corrwith method, you can compute pairwise correlations between a DataFrames columns or rows with another Series or DataFrame.

In [22]:
returns.corrwith(returns.IBM)
Out[22]:
AAPL    0.527378
IBM     1.000000
MSFT    0.593162
GOOG    0.526100
dtype: float64
In [23]:
returns.corrwith(volume)
Out[23]:
AAPL   -0.139780
IBM    -0.115935
MSFT   -0.065446
GOOG   -0.050404
dtype: float64

Passing axis='columns' does things row-by-row instead. In all cases, the data points are aligned by label before the correlation is computed.

2. Unique Values, Value Counts, and Membership¶

Another class of related methods extracts information about the values contained in a one-dimensional Series. To illustrate these, consider this example:

In [24]:
obj = pd.Series(['c', 'a', 'd', 'a', 'a', 'b', 'b', 'c', 'c'])

The first function is unique, which gives you an array of the unique values in a Series:

In [25]:
uniques = obj.unique()
uniques
Out[25]:
array(['c', 'a', 'd', 'b'], dtype=object)

Relatedly, value_counts computes a Series containing value frequencies:

In [26]:
obj.value_counts()
Out[26]:
c    3
a    3
b    2
d    1
dtype: int64

value_counts is also available as a top-level pandas method that can be used with any array or sequence:

In [28]:
pd.value_counts(obj.values, sort=False)
Out[28]:
b    2
a    3
d    1
c    3
dtype: int64

isin performs a vectorized set membership check and can be useful in filtering a dataset down to a subset of values in a Series or column in a DataFrame:

In [30]:
print(obj)
mask = obj.isin(['b', 'c'])

Related to isin is the Index.get_indexer method, which gives you an index array from an array of possibly non-distinct values into another array of distinct values:

In [33]:
to_match = pd.Series(['c', 'a', 'b', 'b', 'c', 'a'])
unique_vals = pd.Series(['c', 'b', 'a'])
pd.Index(unique_vals).get_indexer(to_match)
Out[33]:
array([0, 2, 1, 1, 0, 2], dtype=int64)
